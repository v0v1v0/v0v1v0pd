pvalue R Documentation

Formats an P-value figure for LaTeX

Description

Produces a math-mode formatted string for a P-value.

Usage

pvalue(p, fmt = "$P = %6.4f$")

Arguments

p

The P-value
fmt

A formatting string for LaTeX

Value

A string that will be correctly formatted by LaTeX

Author(s)

J Curran

See Also

Sweave

Examples

p = 0.04
pvalue(p)
