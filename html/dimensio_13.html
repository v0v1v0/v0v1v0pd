MultivariateSummary R Documentation

Summary of Multivariate Data Analysis

Description

A virtual S4 class to store the summary of a multivariate data analysis.

Slots

data

A numeric matrix.

eigenvalues

A numeric matrix.

results

A numeric matrix.

supplement

A logical vector specifying the supplementary points.

margin

An integer.
