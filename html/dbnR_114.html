Generic parameter replacement method for "dbn.fit" S3 objects. Calls bnlearn underneath.

Usage

## S3 replacement method for class 'dbn.fit'
x[[name]] <- value

Arguments

x

the fitted network
name

name of the node to replace its parameters
value

the new parameters

Value

the modified network
