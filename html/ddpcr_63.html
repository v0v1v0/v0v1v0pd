is_range R Documentation

Is the given parameter a range?

Description

Is the given parameter a range?

Usage

is_range(x)

Examples

is_range("C05")            # FALSE
is_range(c("C05", "F05"))  # FALSE
is_range("C05")            # FALSE
is_range("C05, F05")       # TRUE
is_range("C05:F05")        # TRUE
is_range("C05.F05")        # FALSE
