dip6 R Documentation

Dissolution data of a reference and a test batch

Description

A data set containing the simulated dissolution data of one reference batch and one test batch of n = 12 tablets each, i.e. the dissolution profiles of the % drug release observed within a period of 140 minutes. The profiles are simulated to have a kink between 115 and 125 minutes.

Usage

data(dip6)

Format

A data frame with 24 observations and 31 variables:

type

Factor with levels R (Reference) and T (Test)

tablet

Factor with levels 1 to 12 representing individual tablets
