left_equal R Documentation

Compare two characters from left Much useful for arguments input. Case is ignored.

Description

Compare two characters from left Much useful for arguments input. Case is ignored.

Usage

left_equal(x1, x2)

Arguments

x1

one character
x2

the other character

Value

logical

Examples

left_equal('o','OK')
left_equal('ok','O')
left_equal('ok','Ok')
