saveReportTable R Documentation

Output R object using function write.csv.

Description

Output R object using function write.csv.

Usage

saveReportTable(
  x,
  task,
  type,
  ext = "csv",
  subdir = NULL,
  dirCreate = TRUE,
  gzip = FALSE,
  ...
)

Arguments

x

R object to output.
task

Object of class D4TAlinkTask, as created by initTask.
type

Filename type. If the type is an array, the cocatenation of the elements is used with separator"-". Filenames have the form [task name]_[type].[ext]
ext

Filename extension.
subdir

(optional) Subdirectory.
dirCreate

Logical, if TRUE (by default) the directory is created.
gzip

unused.
...

Arguments passed on to utils::write.csv

Value

the file name invisibly.
