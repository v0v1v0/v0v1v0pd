DEEVD-package R Documentation

DEEVD

Description

Two extreme value distributions; Weibull and Gumbel kernel related functions are presented. Weibull and Gumbel present estimated values and plot.Weibull and plot.Gumbel plot the densities. While mean squared error can be calculated by using mse. Further, some related data sets are also presented.

Details

Density Estimation by Extreme Value Distributions

Author(s)

Javaria Ahmad Khan, Atif Akbar.
