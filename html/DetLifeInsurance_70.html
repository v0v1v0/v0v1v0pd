sf R Documentation

Future Value of an Annuity

Description

Calculates the future value of an annuity.

Usage

sf(l = 0, n, i)

Arguments

l

0 for annuity due or 1 for annuity immediate.
n

A numeric value. The number of payments.
i

A numeric value. The interest rate.

Examples

sf(0,12,0.05)
sf(1,23,0.04)
