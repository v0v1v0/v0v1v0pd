names.disto R Documentation

Get names/labels

Description

Get names/labels of the underlying distance storing backend

Usage

## S3 method for class 'disto'
names(x)

Arguments

x

disto object

Value

A character vector

Examples

temp <- stats::dist(iris[,1:4])
dio <- disto(objectname = "temp")
dio
names(dio) <- paste0("a", 1:150)
