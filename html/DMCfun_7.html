calculateCostValueGS R Documentation

calculateCostValueGS

Description

Calculate cost value (fit) using likelihood-ratio chi-square statistic (GS) from correct and incorrect RT data.

Usage

calculateCostValueGS(resTh, resOb)

Arguments

resTh

list containing simulation $sim values (output from dmcSim) for rts_comp, rts_incomp, errs_comp, errs_incomp
resOb

list containing raw observed data (see dmcObservedData with keepRaw = TRUE)

Value

cost value (GS)

Examples

# Example 1:
resTh <- dmcSim()
resOb <- flankerData
resOb <- calculateBinProbabilities(resOb)
cost  <- calculateCostValueGS(resTh, resOb)
