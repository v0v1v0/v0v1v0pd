GlacialTills R Documentation

Glacial Tills

Description

Data from Aitchison (2003)

Usage

GlacialTills

Format

A data frame with 92 observations on the following 5 variables.

Red.Sandstone

a numeric vector

Gray.Sandstone

a numeric vector

Crystalline

a numeric vector

Miscellaneous

a numeric vector

Pcount

a numeric vector

Source

Aitchison, J. (2003). The Statistical Analysis of Compositional Data. The Blackburn Press, Caldwell, NJ.
