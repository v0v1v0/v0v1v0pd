af R Documentation

Present Value of An Annuity

Description

Calculates the present value of an annuity.

Usage

af(l = 0, n, i)

Arguments

l

0 for annuity due or 1 for annuity immediate.
n

A numeric value. The number of payments.
i

A numeric value. The interest rate.

Examples

af(0,10,0.03)
af(1,15,0.05)
