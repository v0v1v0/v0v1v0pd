process_dssat_output_header R Documentation

Extracts run number, model code, experiment code, treatment number and data headline from DSSAT output header

Description

Extracts run number, model code, experiment code, treatment number and data headline from DSSAT output header

Usage

process_dssat_output_header(raw_lines)

Arguments

raw_lines

a character vector that includes the contents of a single tier of data (including version stamp and header information) from a DSSAT output file

Value

a list containing the data headline, model code, experiment code, treatment number, and run number extracted from the raw output

Examples


sample_data_tier <- c(
