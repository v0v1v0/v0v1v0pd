plot_text_options R Documentation

Modify text options of plots

Description

Modify text options of plots

Usage

plot_text_options(
  font_name = "Helvetica",
  title_size = 10,
  label_size = 7,
  legend_size = 10,
  strip_text_y_size = 8
)

Arguments

font_name

Name of font to use
title_size

Font size of x- and y-axis titles
label_size

Font size of x- and y-axis tick labels
legend_size

Font size of legend
strip_text_y_size

Font size of vertical panel text
