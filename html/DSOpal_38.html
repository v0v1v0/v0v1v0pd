dsListPackages,OpalConnection-method R Documentation

List packages

Description

List packages defined in the DataSHIELD configuration.

Usage

## S4 method for signature 'OpalConnection'
dsListPackages(conn)

Arguments

conn

OpalConnection-class class object

Value

A data frame with columns: name, version.

Examples

## Not run: 
con <- dbConnect(DSOpal::Opal(), "server1",
  "username", "password", "https://opal.example.org")
dsListPackages(con)
dsDisconnect(con)

## End(Not run)
