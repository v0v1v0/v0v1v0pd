is_multi_tip_species R Documentation

Checks if a species is represented in the tree as multiple tips and those tips form a monophyletic group (i.e. one species with multiple samples)

Description

Checks if a species is represented in the tree as multiple tips and those tips form a monophyletic group (i.e. one species with multiple samples)

Usage

is_multi_tip_species(phylod, species_label)

Arguments

phylod

A phylo4d object from the package phylobase containing phylogenetic and endemicity data for each species.
species_label

The tip label of the species of interest.

Value

Boolean
