Description

Ignore specific files by git and R build.

Usage

use_ignore(file = NULL, path = NULL)

Arguments

file

character File to ignore.
path

character Path to the file.

Value

invisibly returns 0.

Examples

datapackage_skeleton(name="test",path = tempdir())
use_ignore("foo", ".")
