openDTBin R Documentation

Open a new .dtbin file

Description

Opens a .dtbin file. Prints out an error message if the file is already open.

Usage

openDTBin(path)

Arguments

path

A path name

Value

Nothing returned

Examples

## Not run: 
openDTBin("/tmp/test")
addDTBin("/tmp/test","A",Orange)
infoDTBin("/tmp/test")
closeDTBin("/tmp/test")

## End(Not run)
