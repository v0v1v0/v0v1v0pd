DeletePredictionExplanationsInitialization R Documentation

Delete the prediction explanations initialization for a model.

Description

Delete the prediction explanations initialization for a model.

Usage

DeletePredictionExplanationsInitialization(model)

Arguments

model

An S3 object of class dataRobotModel like that returned by the function GetModel, or each element of the list returned by the function ListModels.

Value

Logical TRUE and displays a message to the user if the delete request was successful; otherwise an error message is displayed.

Examples

## Not run: 
  projectId <- "59a5af20c80891534e3c2bde"
  modelId <- "5996f820af07fc605e81ead4"
  model <- GetModel(projectId, modelId)
  DeletePredictionExplanationsInitialization(model)

## End(Not run)
