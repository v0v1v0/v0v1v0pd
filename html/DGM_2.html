center R Documentation

Mean centers timeseries in a 2D array timeseries x nodes, i.e. each timeseries of each node has mean of zero.

Description

Mean centers timeseries in a 2D array timeseries x nodes, i.e. each timeseries of each node has mean of zero.

Usage

center(X)

Arguments

X

2D array with dimensions timeseries x nodes.

Value

M 2D array.

Examples

data("utestdata")
myts=center(myts)
