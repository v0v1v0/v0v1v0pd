runDTATapp R Documentation

Run Shiny apps included in package DTAT

Description

Run Shiny apps included in package DTAT

Usage

runDTATapp(app)

Arguments

app

Character vector of length 1. Name of app to run.

Value

Invoked for side effect. Runs the named Shiny app.

Examples

if(interactive()){
runDTATapp("Sim33PC")
runDTATapp("TheCost")
}
