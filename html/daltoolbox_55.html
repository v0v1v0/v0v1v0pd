plot points

Usage

plot_points(data, label_x = "", label_y = "", colors = NULL)

Arguments

label_x

label_y

y-axis label
colors

color vector

Value

ggplot graphic

Examples

x <- seq(0, 10, 0.25)
data <- data.frame(x, sin=sin(x), cosine=cos(x)+5)
head(data)

grf <- plot_points(data, colors=c("red", "green"))
plot(grf)
