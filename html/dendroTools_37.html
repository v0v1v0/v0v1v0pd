swit272_daily_temperatures R Documentation

Daily temperatures for swit272 chronology

Description

Mean daily temperature in Celsius for the period 1950 - 2019. This gridded E-OBS data on 0.1Â° regular grid, version 20e. Extracted data is for the grid point with lon = 9.75 and lat = 46.45.

Usage

swit272_daily_temperatures

Format

A data frame with 25414 rows and 2 variables:

date

character string describing date

t_avg

mean temperature
