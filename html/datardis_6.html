torchwood_writers R Documentation

List of Torchwood writers

Description

List of directors for all episodes from Torchwood series (2006-2011)

Usage

torchwood_writers

Format

This data frame contains the following columns:

story_number

Story number

writer

Episode's writer

Source

https://en.wikipedia.org/wiki/List_of_Torchwood_episodes

Examples

library(dplyr)
torchwood_writers |>
    count(writer, sort = TRUE) |>
    head(5)
