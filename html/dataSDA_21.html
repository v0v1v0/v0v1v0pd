iGAP_to_MM R Documentation

iGAP to MM

Description

To convert iGAP files to CSV files.

Usage

iGAP_to_MM(data, location)

Arguments

data

The iGAP file.
location

The location of the symbolic variable in the data.

Value

A CSV data file.

Examples

data(Abalone.iGAP)
Abalone <- iGAP_to_MM(Abalone.iGAP, c(1, 2, 3, 4, 5, 6, 7))
