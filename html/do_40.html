Formatting Replacement

Description

Formatting Replacement

Usage

fmt(x, ...)

Arguments

x

format with slash number and one space, which is like "/1 ". Number means replacement order.
...

values to be passed into x

Value

replaced string

Examples

'whwdzg, ykybnfg'
fmt('/ hwdzg, ykybnfg',
    'w')
fmt('/ h/ dzg, ykybnfg',
    'w')
fmt('/1 h/ dzg, ykybnfg',
    'w')
fmt('/1 h/ dzg, ykybnfg',
    'w','-w-')

fmt('/ h/1 dzg, ykybnfg',
    'w','-w-')

fmt('/1 h/0 dzg, ykybnfg',
    'w','-w-')


'|w|' |> 
    fmt(x = '/ h/ dzg, ykybnfg')


'|w|' |> 
    fmt(x = '/ h/ dzg, ykybnfg',
        '-w-')
'|w|' |> 
    fmt(x = '/ h/1 dzg, ykybnfg',
        '-w-')
