LogisticReg R Documentation

Logistic Regression Function

Description

Logestic CDF(cumulative distribution function).

Usage

LogisticReg(x)

Arguments

x

A nx1 matrix.

Value

y A nx1 matrix. y equals to exp(x)/(1+exp(x)) if y is not NA and 0 else.

Examples

x <- pracma::rand(5,1)
y <- LogisticReg(x = x)
