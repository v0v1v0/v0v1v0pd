calculateBinProbabilities R Documentation

calculateBinProbabilities

Description

Calculate bin probabilities in observed data

Usage

calculateBinProbabilities(resOb, quantileType = 5)

Arguments

resOb

Observed data (see dmcObservedData)
quantileType

Argument (1-9) from R function quantile specifying the algorithm (?quantile)

Value

resOb Observed data with additional $probSubject/$prob table

Examples

# Example 1:
resOb <- flankerData
resOb <- calculateBinProbabilities(resOb)
resOb$prob
