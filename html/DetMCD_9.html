xtractR_M

Description

Internal function. Formats the output for the DetMCD algorithm.

Usage

  xtractR_M(out2,X)

Arguments

out2

A list. Typically the result of a call to DetMCD_RW.
X

a numeric matrix or data frame without missing values.

Value

returns a list.
