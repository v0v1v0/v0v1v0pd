Filter any rows that meeting requirements

Usage

filter_any(dt, operator, cutoff_value)

Arguments

dt

input data.table

operator should be one of l, g. l means less than, g means greater than.
cutoff_value

threshold value

Value

fitlered data table

Examples

data("dt_values")
dataMojo::filter_any(dt_values, operator = "l", .1)
