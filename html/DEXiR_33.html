dexi_bool R Documentation

dexi_bool

Description

Convert a DEXi string to logical. "TRUE", "T" and "1" are interpreted as TRUE, all other strings as FALSE.

Usage

dexi_bool(x)

Arguments

x

character(1).

Value

logical(1).

Examples

dexi_bool("TRUE")
sapply(c("TRUE", "T", "1", TRUE, 1, "FALSE", "F", "0", NULL, NA, NaN), dexi_bool)
