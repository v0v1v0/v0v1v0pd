NA.col.sums R Documentation

Sum of missing value by column

Description

NA is treated as missing value.

Usage

NA.col.sums(data)

Arguments

data

must be dataframe or matrix

Value

sum of missing value by column

Examples

df = data.frame(x=rep(c(1,NA,2,NA,6,NA),10),
                 y=rep(c(1,NA,2),20))
NA.col.sums(df)
