deb_is_tetra R Documentation

Test if an object is of class deb_tetra

Description

Test if an object is of class deb_tetra.

Usage

deb_is_tetra(x)

Arguments

x

An object.

Value

TRUE if object is of class deb_tetra and FALSE if it is not.

Examples

x <- deb_tetra(5, 3, 8, 2)
y <- c(5, 3, 8, 2)

deb_is_tetra(x)
deb_is_tetra(y)
