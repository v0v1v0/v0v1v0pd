debugmode R Documentation

Switching debug mode on and off

Description

The behavior of debugr's main function, dwatch, depends on whether or not the debugr debug mode is activated or not. The debug mode is turned on and off by setting the global option debugr.active to TRUE and FALSE, respectively. This can be accomplished with the debugr_switchOn and debugr_switchOff functions, or manually by running options(debugr.active = TRUE).

Usage

debugr_switchOn()

debugr_switchOff()

debugr_isActive()

Details

When debugr.active = TRUE the debug mode is enabled and dwatch produces debugging outputs to the console (or to a file). In contrast, when the debug mode is disabled, dwatch remains "silent" and no output whatsoever will be shown.

Functions

  • debugr_switchOn: Switches on the global option for debugging

  • debugr_switchOff: Switches off the global option for debugging

  • debugr_isActive: Check if debug mode is currently active or not
