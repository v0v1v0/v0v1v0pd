extend_with R Documentation

Extend an object

Description

Extend an object

Usage

extend_with(object, .class_name, ...)

Arguments

object

A list
.class_name

A class name to add
...

Extra information in the list

Examples

library(purrr)
l <- list(important_number = 42) %>% add_class("my_list")
l %>% extend_with(
  .class_name = "improved_list",
  url = "https://github.com/dynverse/dynverse"
)
l
