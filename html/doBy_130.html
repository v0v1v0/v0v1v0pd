plot_lm R Documentation

Plot linear model object

Description

Plot linear model object

Usage

plot_lm(lm_fit, format = "2x2")

Arguments

lm_fit

An object of class 'lm'
format

The format of the plot (or a list of plots if format is "list")

Examples


m1 <- lm(speed ~ dist, data=cars)
plot_lm(m1)
plot_lm(m1, "2x2")
plot_lm(m1, "1x4")
plot_lm(m1, "4x1")
plot_lm(m1, "list")
