lmmR2 R Documentation

Calculates several measures of fit for Linear Mixed Models based on Lou and Azen (2013) text. Models could be lmer or lme models.

Description

Calculates several measures of fit for Linear Mixed Models based on Lou and Azen (2013) text. Models could be lmer or lme models.

Usage

lmmR2(m.null, m.full)

Arguments

m.null

Null model (only with random intercept effects)
m.full

Full model

Value

lmmR2 class
