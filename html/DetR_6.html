inUMCD R Documentation

Test function for unimcd

Description

Test function for the unimcd used in DetR.

Usage

  inUMCD(x)

Arguments

x

Vector of 2 or more numbers. Should contain no ties.

Value

the value of the unimcd estimator of scale.
