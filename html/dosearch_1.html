dosearch-package R Documentation

Causal Effect Identification from Multiple Incomplete Data Sources

Description

Solves causal effect identifiability problems from arbitrary observational and experimental distributions using a heuristic search. Allows for the presence of advanced data-generating mechanisms. See Tikka et al. (2021) doi:10.18637/jss.v099.i05 for further details.
