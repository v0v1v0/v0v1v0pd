Line.Transect-class R Documentation

Class "Line.Transect" extends Class "Transect"

Description

Class "Line.Transect" is an S4 class detailing a set of transects from a point transect design.

Slots

line.length

the total line length for the transect set

trackline

the total on and off effort trackline length from the start of the first transect to the end of the last

cyclictrackline

the trackline distance plus the distance required to return from the end of the last transect to the beginning of the first
