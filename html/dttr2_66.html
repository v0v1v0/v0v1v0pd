dtt_day_decimal R Documentation

Get Decimal Day Values

Description

Gets decimal day values for date/time vectors.

Usage

dtt_day_decimal(x, ...)

## S3 method for class 'Date'
dtt_day_decimal(x, ...)

## S3 method for class 'POSIXct'
dtt_day_decimal(x, ...)

A date/time vector.
...

Unused.

Value

A numeric vector.
