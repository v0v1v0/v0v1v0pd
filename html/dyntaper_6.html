Id R Documentation

Integral of decay function.

Description

Integral of decay function.

Usage

Id(x, p)

Arguments

x

Input value(s), possibly a vector.
p

Parameter.

Value

Integral of the decay function between 0 and x.

Examples

   Id(2, 0)
   Id(1.5,0.5)
   Id(2.5, 0.5)
   Id(2.5, -0.5)
   for(p in seq(1, -1, -0.5)) curve(Id(x, p), 0, 3, add=(p != 1))
