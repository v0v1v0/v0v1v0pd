to_ts R Documentation

Tabular data to ts() format

Description

transform data in 'x' corresponding to dates in 'dates' to ts() format

Usage

to_ts(x, dates)

Arguments

x

data
dates

dates

Value

data in ts() format

Examples

x <- c(1,2,3,4)
dates <- as.Date(c("2020-1-1","2020-2-1","2020-3-1","2020-4-1"))
to_ts(x, dates)
