simul.data R Documentation

Simulated Data Set

Description

Simulated data set with 100 persons, 10 items and 3 (standardized) covariates. Items 1, 2 and 3 are DIF items.

Usage

data(simul.data)

Format

Item1

Item 1, DIF item

Item2

Item 2, DIF item

Item3

Item 3, DIF item

Item4

Item 4, non-DIF item

Item5

Item 5, non-DIF item

Item6

Item 6, non-DIF item

Item7

Item 7, non-DIF item

Item8

Item 8, non-DIF item

Item9

Item 9, non-DIF item

Item10

Item 10, non-DIF item

CovBin1

Binary covariate (standardized)

CovBin2

Binary covariate (standardized)

CovMet

Metric covariate (standardized)

References

Tutz, Gerhard and Schauberger, Gunther (2015): A Penalty Approach to Differential Item Functioning in Rasch Models, Psychometrika, 80(1), 21 - 43

Examples

## Not run: 
data(simul.data)

Y <- simul.data[,1:10]
X <- simul.data[,11:13]

m1 <- DIFlasso(Y = Y, X = X, trace = TRUE) 
print(m1)
plot(m1)

m2 <- refitDIFlasso(m1)
print(m2)
plot(m2)

## End(Not run)
