as.objlist R Documentation

Generate objective list from numeric vector

Description

Generate objective list from numeric vector

Usage

as.objlist(p)

Arguments

p

Named numeric vector

Value

list with entries value (0), gradient (rep(0, length(p))) and hessian (matrix(0, length(p), length(p))) of class obj.

Examples

p <- c(A = 1, B = 2)
as.objlist(p)
