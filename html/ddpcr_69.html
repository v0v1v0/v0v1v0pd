local_maxima R Documentation

Get the indices of the local maxima in a list of numbers

Description

Get the indices of the local maxima in a list of numbers

Usage

local_maxima(x)

Arguments

x

Vector of numbers.

Value

A vector containing the indices of the elements that are local maxima in the given input.

Examples

local_maxima(c(1, 5, 3, 2, 4, 3))
