deb_is_decimal R Documentation

Test if an object is of class deb_decimal

Description

Test if an object is of class deb_decimal.

Usage

deb_is_decimal(x)

Arguments

x

An object.

Value

TRUE if object is of class deb_decimal and FALSE if it is not.

Examples

x <- deb_decimal(c(5.25, 3.825, 8.5))
y <- c(5.25, 3.825, 8.5)

deb_is_decimal(x)
deb_is_decimal(y)
