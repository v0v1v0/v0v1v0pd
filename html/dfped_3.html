alpha1AGage R Documentation

Concentration of alpha1-acid glycoprotein according to age.

Description

Compute the value of alpha1-acid glycoprotein (alpha1AG) concentration (g/L) according to age (year) for children.

Usage

alpha1AGage(age)

Arguments

age

The age of children.
