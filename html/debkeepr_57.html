deb_is_lsd R Documentation

Test if an object is of class deb_lsd

Description

Test if an object is of class deb_lsd.

Usage

deb_is_lsd(x)

Arguments

x

An object.

Value

TRUE if object is of class deb_lsd and FALSE if it is not.

Examples

x <- deb_lsd(5, 3, 8)
y <- c(5, 3, 8)

deb_is_lsd(x)
deb_is_lsd(y)
