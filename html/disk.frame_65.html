is_disk.frame R Documentation

Checks if a folder is a disk.frame

Description

Checks if a folder is a disk.frame

Usage

is_disk.frame(df)

Arguments

df

a disk.frame or directory to check

Examples

cars.df = as.disk.frame(cars)

is_disk.frame(cars) # FALSE
is_disk.frame(cars.df) # TRUE

# clean up cars.df
delete(cars.df)
