getLow, getUp functions of package distr

Description

getLow, getUp return lower and upper endpoint of a distribution â€” truncated to lower/upper TruncQuantile if infinite; in case of an object of class "LatticeDistribution" with infinite lattice length, we search for the smallest/largest point in the lattice which is returned by succesive halving of x=0.5 in q(object)(x, lower.tail) for lower.tail TRUE resp. false.
