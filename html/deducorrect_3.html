applyFix R Documentation

Apply flips and swaps to a record.

Description

Apply flips and swaps to a record.

Usage

applyFix(flips, swaps, r)

Arguments

flips

Index vector in r. r will be sign-flipped at flips
swaps

nx2 matrix denoting value swaps in r.
r

numerical record.

Value

r, with flips and swaps applied
