get_from_list R Documentation

Get from list

Description

Retrieve object 'what' from 'lst'

Usage

get_from_list(lst, what)

Arguments

lst

list
what

object to retrieve (by name or index)

Value

Element of the list indicated

Examples

get_from_list(list("a" = "alpha", "b" = c(1,2,3)), "a") # "alpha"
