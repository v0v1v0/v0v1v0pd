tock R Documentation

Stop timer

Description

Stop timer and write the computation in seconds since the last call of tick().

Usage

tock(txt = "")

Arguments

txt

optional custom text to be displayed

Value

None

Author(s)

Danail Obreschkow

See Also

tick

Examples


tick('Sum 10 million random numbers')
x = sum(runif(1e7))
tock()
