round.date_yq R Documentation

Rounding of date_xx

Description

Rounds a date_xx to the first unit of the current year, or the first unit of the next year.

Usage

## S3 method for class 'date_yq'
round(x, digits = NULL)

## S3 method for class 'date_ym'
round(x, digits = NULL)

## S3 method for class 'date_yw'
round(x, digits = NULL)

## S3 method for class 'date_xx'
ceiling(x)

## S3 method for class 'date_xx'
floor(x)

Arguments

x

any date_xx object
digits

ignored, only there for compatibility with base::round()

Value

a date_xx of the same subclass as x

Examples

round(date_yq(2018, 2))
round(date_yq(2018, 3))
round(date_ym(2018, 6))
round(date_ym(2018, 7))
round(date_yw(2018, 26))
round(date_yw(2018, 27))
