norm_max R Documentation

norm_max

Description

Normalize values so that max(values) == max.

Usage

norm_max(values, max = 1)

Arguments

values

A numeric vector.
max

numeric(1).

Value

values normalized so that max(result) == max. Returns unchanged values when max(values) == 0.

See Also

norm_none(), norm_sum()

Examples

norm_max(c(0, 0.5, 0.7))
