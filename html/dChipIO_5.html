readCdfBinHeader R Documentation

Reads the file header of a dChip CDF.bin file

Description

Reads the file header of a dChip CDF.bin file.

Usage

readCdfBinHeader(con, ...)

Arguments

con

A connection or a character filename.
...

Not used.

Value

Returns a list structure containing the file header.
