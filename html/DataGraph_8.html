openDTable R Documentation

Open a .dtable file so you can write into it using addDTable

Description

Saves the data frame to a file

Usage

openDTable(path)

Arguments

path

A path name

Value

Nothing returned

Examples

## Not run: 
openDTable("/tmp/test")
addDTable("/tmp/test",Orange)
closeDTable("/tmp/test")

## End(Not run)
