Dist2CP R Documentation

Dist2CP

Description

Transforms a cube of distance matrices into a cube of cross-product matrices.

Usage

Dist2CP(D3)

Arguments

D3

the cube of distance matrices.

Value

the cube of cross-product matrices.

Examples


D3 <- array(c(0, 1, 2, 1, 0, 1, 2, 1, 0, 
              0, 3, 3, 3, 0, 3, 3, 3, 0), 
            dim = c(3, 3, 2))
Dist2CP(D3)
