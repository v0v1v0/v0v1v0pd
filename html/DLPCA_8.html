Iris R Documentation

Iris

Description

Iris data set

Usage

data("Iris")

Format

A data frame with 150 observations on the following 5 variables.

Sepal.length

a numeric vector

Sepal.width

a numeric vector

Petal.length

a numeric vector

Petal.width

a numeric vector

Species

a character vector

Details

It contains 150 samples with 5 variables

Source

Gaspar peninsula in Canada

Examples

data(Iris)
