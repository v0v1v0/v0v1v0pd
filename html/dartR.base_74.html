gl.report.basics R Documentation

Basic statistics for a genlight object

Description

Calculates basic statistics for a genlight object.

Usage

gl.report.basics(x, verbose = NULL)

Arguments

x

Name of the genlight object [required].
verbose

Verbosity: 0, silent or fatal errors; 1, begin and end; 2, progress log; 3, progress and results summary; 5, full report [default 2, unless specified using gl.set.verbosity].

Author(s)

Custodian: Arthur Georges (Post to https://groups.google.com/d/forum/dartr)

See Also

Other unmatched report: gl.allele.freq(), gl.report.diversity(), gl.report.heterozygosity(), gl.report.polyploid_heterozygosity()

Examples

gl.report.basics(platypus.gl)
