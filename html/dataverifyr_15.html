write_rules R Documentation

Read and write rules to a yaml file

Description

Read and write rules to a yaml file

Usage

write_rules(x, file)

read_rules(file)

Arguments

x

a list of rules
file

a filename

Value

the filename invisibly

Functions

  • read_rules(): reads a ruleset back in

Examples

rr <- ruleset(
  rule(mpg > 10),
  rule(cyl %in% c(4, 6, 8))
)
file <- tempfile(fileext = ".yml")
write_rules(rr, file)
