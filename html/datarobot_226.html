parseRFC3339Timestamp

Description

The DataRobot APIs returns dates in RFC 3339 format.

Usage

parseRFC3339Timestamp(timestampstring)

Arguments

timestampstring

character. Timestamp in RFC 3339 format.

Value

The input timestamp as a POSIXt
