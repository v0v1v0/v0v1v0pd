dm_nrow R Documentation

Number of rows

Description

Returns a named vector with the number of rows for each table.

Usage

dm_nrow(dm)

Arguments

dm

A dm object.

Value

A named vector with the number of rows for each table.

Examples


dm_nycflights13() %>%
  dm_filter(airports = (faa %in% c("EWR", "LGA"))) %>%
  dm_nrow()
