Change data type

Description

Change data type

Usage

factor.it(x) <- value

factor.it(x, value)

numeric.it(x, value)

numeric.it(x) <- value

Arguments

x

dataframe
value

column names

Value

factor or numeric columns in a dataframe

Examples

str(mtcars)
factor.it(mtcars,c("cyl", "vs", "am", "gear"))
factor.it(mtcars)=c("cyl", "vs", "am", "gear")
str(mtcars)

numeric.it(mtcars,c("cyl", "vs", "am", "gear"))
numeric.it(mtcars)=c("cyl", "vs", "am", "gear")
str(mtcars)
