class_scope R Documentation

Calling Class Scope

Description

Gets the exposed public field scope of a R6 class. Used for evaluating cls field execution scopes.

Usage

class_scope(cls)

Arguments

cls

R6 class to export.

Value

system context for evaluating fmt_metric objects.
