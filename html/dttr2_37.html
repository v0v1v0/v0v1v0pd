dtt_date_add_time R Documentation

Add Time to Date

Description

Adds times to Dates vector and sets timezone in a single function.

Usage

dtt_date_add_time(x, time, tz = dtt_default_tz())

Arguments

x

A Date vector.
time

A hms vector of the time.
tz

A string of the time zone.

Value

A POSIXct vector.

See Also

Other add: dtt_add_units()

Examples

dtt_date_add_time(
  as.Date("2001-03-05"),
  hms::as_hms("06:07:08"),
  tz = "Etc/GMT+9"
)
