is_email_address R Documentation

Test if a string is an e-mail address.

Description

Test if a string is an e-mail address.

Usage

is_email_address(.x)

Arguments

.x

A character vector.

Value

A logical value indicating if that string is an e-mail address.

Examples

# Examples
is_email_address("hello") # FALSE
is_email_address("hello@world.edu") # TRUE
