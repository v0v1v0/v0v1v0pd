%contains% R Documentation

Does a table contain a value

Description

Does a table contain a value

Usage

table %contains% y

contains(table,y)

Arguments

table

a table of values
y

a value

Details

Literally %in% in reverse order, just for convenience.

Value

a logical vector of the same length as y indicating if y is in table, i.e. the table contains y.
