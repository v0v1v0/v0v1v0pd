clu_tune R Documentation

Clustering Tune

Description

Clustering Tune

Usage

clu_tune(base_model)

Arguments

base_model

base model for tuning

Value

a clu_tune object.

Examples

data(iris)

# fit model
model <- clu_tune(cluster_kmeans(k = 0))
ranges <- list(k = 1:10)
model <- fit(model, iris[,1:4], ranges)
model$k
