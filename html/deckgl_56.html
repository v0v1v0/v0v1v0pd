Create a contour definition

Description

Create a contour definition

Usage

use_contour_definition(
  threshold = 1,
  color = c(255, 255, 255),
  stroke_width = 1
)

Arguments

threshold

The threshold value used in contour generation.
color

The RGB color array used to render contour lines.
stroke_width

The width of the contour lines in pixels.
