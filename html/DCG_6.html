GetSim R Documentation

GetSim get similarity matrix from a distance matrix

Description

GetSim get similarity matrix from a distance matrix

Usage

GetSim(D, T)

Arguments

D

A distance matrix
T

Temperature. temperatureSample

Details

the similarity matrix is calculated at each temperature T.
