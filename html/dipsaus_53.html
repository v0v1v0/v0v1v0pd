do_aggregate R Documentation

Make aggregate pipe-friendly

Description

A pipe-friendly wrapper of aggregate when using formula as input.

Usage

do_aggregate(x, ...)

Arguments

x

an R object
...

other parameters passed to aggregate

Value

Results from aggregate

See Also

aggregate

Examples


library(magrittr)
data(ToothGrowth)

ToothGrowth %>%
  do_aggregate(len ~ ., mean)
