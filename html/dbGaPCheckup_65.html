Example data set and data dictionary with intentional errors.

Usage

data(ExampleN)

R data file that contains two objects:

DD.dict.N

Data dictionary

DS.data.N

Data set

DD.path <- system.file("extdata", "3b_SSM_DD_Example2e.xlsx", package = "dbGaPCheckup", mustWork=TRUE)
