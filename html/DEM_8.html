magic R Documentation

Magic

Description

The magic data

Usage

data("magic")

Format

A data frame with 19020 observations on the following 11 variables.

fLength

a numeric vector

fWidth

a numeric vector

fSize

a numeric vector

fConc

a numeric vector

fConc1

a numeric vector

fAsym

a numeric vector

fM3Long

a numeric vector

fM3Trans

a numeric vector

fAlpha

a numeric vector

fDist

a numeric vector

class

a character vector

Details

The magic data set is given by MAGIC project, and described by 11 features.

Source

The magic data set is from the UCI database.

References

J. Dvorak, P. Savicky. Softening Splits in Decision Trees Using Simulated Annealing. Proceedings of ICANNGA 2007, Warsaw, Part I, LNCS 4431, pp. 721-729.

Examples

data(magic)
