createTask R Documentation

Create a task.

Description

Create a task.

Usage

createTask(
  project,
  package,
  taskname,
  sponsor = getTaskSponsor(),
  author = getTaskAuthor()
)

Arguments

project

Project name.
package

Package name.
taskname

Task name.
sponsor

Sponsor name, default set by setTaskSponsor.
author

Author name, system username by default.

Value

An D4TAlinkTask object
