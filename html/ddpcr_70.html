local_minima R Documentation

Get the indices of the local minima in a list of numbers

Description

Get the indices of the local minima in a list of numbers

Usage

local_minima(x)

Arguments

x

Vector of numbers.

Value

A vector containing the indices of the elements that are local minima in the given input.

Examples

local_minima(c(1, 5, 3, 2, 4, 3))
