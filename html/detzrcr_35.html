plot_ecdf R Documentation

Plot ecdf

Description

Plot ecdf for U-Pb age or Lu-Hf model age data.

Usage

plot_ecdf(
  dat,
  mult_ecdf = FALSE,
  column = "age",
  conf = FALSE,
  guide = TRUE,
  alpha = 0.05
)

Arguments

dat

data.frame
mult_ecdf

logical, plot several ecdfs in same plot
column

which column to use
conf

logical, plot confidence bands
guide

logical, show legend
alpha

alpha-level used for confidence bands
