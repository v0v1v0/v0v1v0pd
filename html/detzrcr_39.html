Description

Plot quantiles

Usage

plot_quantiles(
  dat,
  column = "t_dm2",
  conf = FALSE,
  alpha = 0.05,
  type = 8,
  guide = TRUE,
  mix = FALSE,
  mix_data = NULL
)

Arguments

dat

data.frame
column

which column in data.frame to use
conf

logical, plot confidence interval
alpha

alpha-level
type

type of quantile calculation (passed on to stats::quantile)
guide

logical, show legend?
mix

logical, add mixing model
mix_data

mixing model data
