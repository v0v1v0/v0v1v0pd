is.fmdsd R Documentation

Class fmdsd

Description

Tests if its argument is an object of class fmdsd (see Details of the function fmdsd).

Usage

is.fmdsd(x)

Arguments

x

object to be tested.

Value

TRUE if its argument is of class fmdsd, and FALSE otherwise.
