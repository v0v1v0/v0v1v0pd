Nchar R Documentation

Number of Characters

Description

Number of Characters

Usage

Nchar(x)

Arguments

x

can be number, strings, verctors, dataframe or matrix.

Value

number of characters in each location

Examples

Nchar("abcd")
Nchar(c("abc","gjh"))
df = data.frame(
  a = c(1,12,12.3),
  b = c("a","ab","abc")
)
Nchar(df)
