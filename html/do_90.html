NA.whole.prob R Documentation

Proportion of missing value in the whole dataframe

Description

NA is treated as missing value.

Usage

NA.whole.prob(data)

Arguments

data

must be dataframe or matrix

Value

proportion of missing value in the whole dataframe

Examples

df = data.frame(x=rep(c(1,NA,2,NA,6,NA),10),
                 y=rep(c(1,NA,2),20))
NA.whole.prob(df)
