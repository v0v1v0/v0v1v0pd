Idd R Documentation

Double integral of decay function.

Description

Double integral of decay function.

Usage

Idd(x, p)

Arguments

x

Input value(s), possibly a vector.
p

Parameter.

Value

Iterated integral of the decay function between 0 and x, that is, the integral of Id(x, p).

Examples

   Idd(2, 0)
   Idd(1.5,0.5)
   Idd(2.5, -1)
   Idd(2.5, -0.5)
   for(p in seq(1, -1, -0.5)) curve(Idd(x, p), 0, 3, add=(p != 1))
