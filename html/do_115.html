Remove elements by number of characters

Description

Remove elements by number of characters

Usage

rm_nchar(x, least, most)

Arguments

x

one vector
least

least number of characters
most

most number of characters

Value

removed vector

Examples

x <- c('a','abc','abcd',NA)
rm_nchar(x,least = 1)
rm_nchar(x,most = 4)
rm_nchar(x,least = 1, most = 4)
