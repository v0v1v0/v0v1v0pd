is_date_time R Documentation

Is Date/Time

Description

Tests whether an object is a Date, POSIXct, or hms vector.

Usage

is.POSIXct(x)

is_date_time(x)

is.Date(x)

is_date(x)

is.hms(x)

is_time(x)

Arguments

x

An object

Value

A flag indicating whether x inherits from Date, POSIXct or hms.

See Also

Other is: dtt_is_date(), dtt_is_date_time()
