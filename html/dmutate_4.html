as_idata R Documentation

Create individual data frame from a covset object

Description

Create individual data frame from a covset object

Usage

as_idata(.covset, .n)

Arguments

.covset

a covset object
.n

number of IDs to simulate

Examples

cov1 <- covset(Y ~ rbinomial(0.2), Z ~ rnorm(2,2))

as_idata(cov1, 10)
